This function uses recursion

1. The function takes in the the argument, checks if it is a dictionary.
2. If it is, it iterates through each of the values, appending them at the 
end of the string argument and separating them through commas.
3. While the program is doing this, it is also checking if the nested values
are also dictionaries.
4. If the nested values are also dictionaries, call the function with the 
dictionary and appended string as the arguments
5. When finished, return the string value
